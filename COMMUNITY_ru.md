# Сообщество (RU)

## Что в OSS-наборе

- Установка: `SETUP.cmd`, `init.mjs`, `install` / `repair` / `doctor`
- Профили minimal → founder — [PROFILE_COMPARISON.md](meta/docs/PROFILE_COMPARISON.md)
- Философия, карта навигации, правила Cursor
- Режим приватности: `--gitignore-kit full`
- Бенчмарки — [BENEFITS_AND_METRICS_ru.md](meta/docs/BENEFITS_AND_METRICS_ru.md), [benchmarks/RUNBOOK.md](benchmarks/RUNBOOK.md)

**Не входит:** рантайм платформы AgentStack, хостинг.

## Как помочь

1. Звезда и watch на [agentstacktech/genetic-ai-starter](https://github.com/agentstacktech/genetic-ai-starter)
2. Discussions — вопросы и витрина проектов
3. Good first issues на GitHub
4. Добавить проект в [SHOWCASE.md](SHOWCASE.md)

## Первый запуск с полным gitignore

После `install --gitignore-kit full` файлы kit в `.gitignore`. Уже закоммиченные пути — `git rm -r --cached` (см. `docs/ai/OPERATIONS.md` в проекте).

## Пасхалки

`doctor --beacon` после успешной установки. Подробности для мейнтейнеров: [meta/docs/EASTER_EGGS.md](meta/docs/EASTER_EGGS.md).
