## Core insight (Lance, paraphrased)

**Ideal system:** as close as possible to **100% function at 0% waste** — knowing real systems trade along a frontier, not a fantasy point.

Minimalism is the **visible** consequence: we optimize what the **consumer of information** must parse — human, AI agent, robot, integrator.

| Consumer | Bottleneck we respect |
|----------|------------------------|
| **Human** | Structure, familiar terms, no random `dfjgh459…` blobs; ≤ quick-scan chunk size |
| **AI agent** | Genetic tags, `AI_INDEX.md` hot files, clusters — not 400-file grep |
| **Integrator** | Stable SDK facades, OpenAPI, MCP matrix |
| **Robot / automation** | Same JSON semantics as SPA ([8DNA §7](https://github.com/agentstacktech/AgentStack/tree/main/docs/8DNA_UNIFIED_REFERENCE.md)) |

## Two axes (the “formula”)

```
        function delivered
              ↑
              │    ● ideal (aim here)
              │
              └──────────────→ cost (bytes, CPU, attention, storage)
```

- **More function without cost** — creation moves (unified table, client offload).  
- **Same function, less cost** — minimalism moves (one endpoint, native semantics).  
- **Wrong quadrant** — more cost *and* more confusion (second cache “just in case”).
