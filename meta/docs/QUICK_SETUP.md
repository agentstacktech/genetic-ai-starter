# Быстрая установка — Genetic AI Starter Kit

**Зачем kit:** карта навигации + правила Cursor + genes — см. [README.md](../../README.md) / [README.en.md](../../README.en.md).

## 3 шага

1. Установите [Node.js 18+](https://nodejs.org).
2. Откройте папку **genetic-ai-starter** (не ваш проект).
3. Запустите **`SETUP.cmd`** (Windows) или `node scripts/init.mjs`.

В мастере:

| Вопрос | Подсказка |
|--------|-----------|
| Куда установить | «Текущая папка» — если вы уже в `cd C:\Projects\MyApp`, иначе укажите путь |
| Имя проекта | Как в README / Cursor |
| Domain | `app` для обычного приложения |
| Профиль | **Стандарт** для большинства; **AgentStack (полный)** для MCP/SDK |
| Git | **Не коммитить** — если kit не должен попасть в remote (полный блок в `.gitignore`) |

Готово → откройте проект в Cursor → `AGENTS.md` (файлы остаются локально; при полном gitignore не уйдут в `git push`).
