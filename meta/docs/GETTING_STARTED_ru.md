# Быстрый старт

**EN:** [GETTING_STARTED.md](GETTING_STARTED.md)

**Рекомендуется (submodule):**

1. `git submodule add https://github.com/agentstacktech/genetic-ai-starter.git tools/genetic-ai-starter`
2. `node tools/genetic-ai-starter/scripts/bootstrap-standard.mjs --target . --project-name "My App" --domain app`
3. Заполните `docs/ai/AI_NAVIGATION_MAP.md`
4. `node tools/genetic-ai-starter/scripts/doctor.mjs --target .`

**Без submodule:** `npx @agentstack/genetic-ai-starter init --yes --target ./my-app --profile standard --project-name "My App" --domain app`
