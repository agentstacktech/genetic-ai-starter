## Quick start

**Install (30 seconds):**

```bash
npx @agentstack/genetic-ai-starter@0.4.11 init --yes --target ./my-app --profile standard --project-name "My App" --domain app
```

**Windows:** double-click `SETUP.cmd` in the kit folder.

**After install:** read `AGENTS.md` and `docs/ai/AI_NAVIGATION_MAP.md` in your project.

**Health check:**

```bash
node node_modules/@agentstack/genetic-ai-starter/scripts/doctor.mjs --target ./my-app
```
